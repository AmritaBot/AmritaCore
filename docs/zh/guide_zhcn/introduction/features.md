# 主要功能概述

## 1.2.1 7个核心功能

1. **基于流的设计**: 所有消息输出均为异步流式设计，支持实时响应
2. **Cookie安全检测**: 内置cookie安全检测功能，保障会话安全
3. **供应商无关机制**: 数据类型和对话管理不依赖特定供应商，具备高可移植性
4. **MCP客户端支持**: 扩展机制中集成MCP客户端，增强系统扩展能力
5. **事件驱动架构**: 完整的事件系统，实现灵活和响应式的Agent行为
6. **工具集成框架**: 健壮的系统用于集成外部工具和服务
7. **高级记忆管理**: 完善的上下文处理，具有自动摘要和Token优化功能

## 1.2.2 架构优势

AmritaCore的架构设计充分考虑了模块化。核心组件可以独立使用或组合使用，从而实现：

- 针对特定用例的轻松定制
- 渐进式功能采用
- 组件间的耦合度降低
- 简化的测试和维护

## 1.2.3 应用场景示例

**场景1: 客服机器人**
能够访问知识库、处理订单并将复杂问题升级给人类客服代表的机器人。

**场景2: 研究助手**
能够从多个来源收集信息、分析文档并综合报告的Agent。

**场景3: 工作流自动化**
协调各种工具和服务以自动化业务流程的智能系统。

这些场景受益于AmritaCore的事件驱动架构、工具集成功能和强大的记忆管理。
