# ToolsManager

The ToolsManager class manages registered tools.

## Description

The ToolsManager class is responsible for managing and maintaining all registered tools in AmritaCore. It provides functionalities for registering tools, querying available tools, invoking tools according to requests, and maintaining metadata information about tools.

The ToolsManager may have the following capabilities:

- Registering new tool functions
- Querying available tools
- Invoking corresponding tools based on requests
- Maintaining metadata information about tools

This class plays a core role in AmritaCore's tool calling system, enabling the AI model to discover and use various external tools to accomplish complex tasks.
