# AmritaCore - Agent Core Module

From [Proj.Amrita](https://github.com/AmritaBot/Amrita)

## Overview

**AmritaCore** is the intelligent agent core module extracted from Proj.Amrita. A lightweight, extensible foundation for building AI-powered agents with advanced capabilities.

> **Documentation under construction** - More detailed documentation coming soon!

## Key Features

- **Lightweight & Modular**: Minimal dependencies, easy integration
- **Streaming Responses**: Real-time, asynchronous message processing
- **Smart Context Management**: Automatic summarization and memory management
- **Tool Integration**: Extend capabilities with custom tools
- **Event-Driven**: Pluggable architecture for customization
- **Provider Agnostic**: Works with multiple LLM providers
- **Security Focused**: Built-in safety and validation mechanisms
